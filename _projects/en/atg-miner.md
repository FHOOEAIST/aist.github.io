## Attribute grammar - Miner

In this project, analysis methods for the extraction of grammars from software systems are to be researched\. As an extension to known analysis methods, semantic aspects in particular are to be taken into account in order to be able to generate automatically attributed grammars \(ATG\) by analyzing software systems\. The extension by semantic aspects promises an improvement of grammar\-based test procedures and opens beyond that new application type in the software development \(e\.g\. Redokumenation\), in which grammars did not play a role so far\. For the extraction of ATGen, in contrast to existing approaches that use dynamic analysis, modern symbolic analysis methods are to be used that are particularly suitable with respect to the extraction of semantic aspects\.
