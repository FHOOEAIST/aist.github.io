## Project CGM Clinical

The goal of KIMBO is to create a concept and prototype implementation of a generalized workflow system to plan, execute and document interdisciplinary medical boards\. In the project a number of different existing boards, such as tumor boards, are researched and generalized into a concept that allows specification and execution of any board types\. The goal is to enable stronger cooperation between medical personnel due to technical solutions, which should increase treatment quality\. Furthermore a prototype based on the healthcare standard HL7 FHIR, should allow users to define their own boards, based on their desired workflow, using mobile devices\.
