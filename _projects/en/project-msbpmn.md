## Project MSBPMN

In the project MSBPMN a standards\-based model transformation between BPMN 2\.0 and HL7 FHIR has to be developed\. The goal is to define clinical guidelines and treatment plans in BPMN and to automatically generate FHIR resources, which are then used in the IT systems for process control and documentation\. The description and implementation of the model transformation itself shall become part of the HL7 standard and is therefore internationally coordinated\. This project will make a significant contribution to the digitalization \(of processes\) of evidence\-based medicine\.
