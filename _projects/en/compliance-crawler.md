## Compliance Crawler

In this project co\-operation, a web crawler is to be developed for configurable websites to evaluate/recognise changes \(legal changes, changed sights\)\. The configured websites are cyclically checked for changing content\. If changes of a content\-related nature are recognised, a structured notification is sent to those responsible\. The changed content \(tariffs, laws, points of interest\) must be prepared in text form in a structured format so that other pages can subsequently be automatically adapted in line with the changes detected\.
