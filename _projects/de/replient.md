## Replient

Innerhalb der Projektkooperation mit Doppel N Marketing GmbH wird an der Entwicklung einer innovativen Software geforscht, die mittels KI\-Technologien die Verwaltung von Kommentaren auf Social Media Plattformen optimieren soll\. Die Software soll in der Lage sein, dem User drei markenspezifische, KI\-generierte Antworten auf neue Social Media Kommentare anzubieten\. Dazu analysiert sie die Website des Unternehmens und lernt aus historischen Kommentaren\. Die Software soll signifikante Zeit\- und Kosteneinsparungen für User ermöglichen, indem es eine effiziente, automatisierte Antwortfindung und ein tiefgreifendes Verständnis der Kundeninteraktionen gewährleistet\. Ein besonderer Fokus dieses Projekts liegt auf der Entwicklung einer Sentiment\-Analyse aller historischen Kommentare, um imn Form eines Dashboard den Usern einen verbesserten Überblick zu bieten und das Antwort\-Prompting darauf basierend zu optimieren\. Letztendlich soll jedes historische und neue Kommentar der Social Media Seite des Users mittels künstlicher Intelligenz einer der zuvor definierten Kategorien zugewiesen werden\.
